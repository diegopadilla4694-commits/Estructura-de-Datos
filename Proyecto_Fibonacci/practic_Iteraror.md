##Iterators Practice

This Python practice exercise used iterators, which are used to count and return values ​​from a given list. The iterator counts and returns the values, acting as an assistant.

--Practice Development
The program uses a counter defined with `self` within the `Num` class. This counter tells the iterator how many iterations are needed to sum each object. A `for` loop is used for this reason. The magic happens in the `def __next__` class, where the value is returned along with the sum. This exercise helped us understand how to use iterators: you ask them for a value, they give it back, and so on.
